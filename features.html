Features

The BuildingSystems library has a flexible modelling architecture and can be used for a large spectrum of modelling and simulation purposes.

General features

  • Supports the present Modelica version 3.3
  • Its a Open Source library (Modelica License Version 2)
  • Uses the Annex 60 model library as a core layer
  • Includes databases for weather data, building materials, building constructions and energy plant componenets

Building simulation

  • Hierarchical modelling approach (models of building elements, single zones, multi-zone buildings, districts)
  • Zones and building models with a different levels of detail (low-order models, 1D, 3D)
  • Thermal and hygrothermal building models

Room simulation

Thermal room simulation model which uses the 3 dimensional modelling approach of the library

Building simulation

Multi-zone thermal building model based on the 1 dimensional modelling approach of the library

District simulation

District model which consists of al lot of low-order building models of the library

Energy plant simulation

  • HVAC system modelling (heating systems, air-conditioning systems)
  • Renewable energy systems (solar thermal systems, photovoltaics systems)
  • District heating and cooling grids

Integrated building and plant simulation

  • Full coupled system models, which consists of one or several building models and energy supply system models
  • Examples: Solar thermal heating system, Photovoltaic cooling systems
